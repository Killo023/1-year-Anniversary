# Valentine's Day Landing Page

An immersive, high-impact Valentine's landing page built with React, Tailwind CSS, and Framer Motion.

## Features

- **Hero Section**: Massive "SATURDAY" heading with overlay text and Aurora gradient background
- **Glassmorphism UI**: Semi-transparent itinerary cards with backdrop blur effects
- **Interactive Map Path**: Decorative SVG path that draws itself as you scroll through "The Wilds" section
- **Full-Screen CTA**: Darkening background with glowing "Will you be my Valentine?" text
- **Magnetic Buttons**: Buttons that subtly pull toward your cursor
- **Sticky Navigation**: Ark Digital logo in top-left corner
- **Floating Verse**: James 1:17 verse with parallax scroll effect

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- **React 18** - UI framework
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animation library
- **Google Fonts** - Inter (sans-serif) and Playfair Display (serif)

## Design Features

- **Aurora Gradient**: Animated mesh gradient background
- **Glassmorphism**: Frosted glass effect with backdrop blur
- **Mix-Blend-Mode**: Editorial-style text blending
- **Scroll Animations**: Reveal animations triggered by scroll position
- **Parallax Effects**: Floating elements that move at different speeds

## Customization

- Update colors in `tailwind.config.js`
- Modify content in component files
- Adjust animations in Framer Motion components
- Replace logo in `Navbar.jsx`
